🧮 Health Metrics Calculator with Risk Bands

A collaborative React-based health companion built by Aditya Kumar and Abhishek Kumar. This app calculates key health metrics like BMI, BMR, TDEE, Waist-to-Height Ratio, and Body Fat %, and provides instant, color-coded risk feedback with personalized health tips. Designed for mobile-first use, fast performance, and interview-ready polish.

🚀 Features

- ✅ BMI, BMR, TDEE, Waist-to-Height Ratio, Body Fat % calculators
- 🎯 Risk Band assignment with color-coded feedback
- 📊 Chart.js visualizations (BMI trend, risk distribution)
- 🗣️ Voice input for accessibility
- 📄 PDF export of health summary
- 🎯 Goal tracker with progress bar
- 🌗 Theme toggle (light/dark/company branding)
- 🧠 Personalized health tips based on metrics

---

🛠️ Tech Stack

- Frontend: Html , Css , Tailwind CSS, JavaScript , React
- Visualization: Chart.js
- APIs: Web Speech API (for voice input)
- Deployment: Netlify / Vercel
